Welcome to phoxpy's documentation!
==================================

This is free open source library that implements client interface to proprietary
`RosLabSystems`_ software `Laboratory Information System`_.

Our goal is to provide handy and flexible tools to develop custom clients and
automatisation utilities. However, you still have at least one active license
to make your client works.

-----------
Source Code
-----------

Source code is available at Google Code http://code.google.com/p/phoxpy::

  hg clone https://phoxpy.googlecode.com/hg phoxpy
